Para importar:

1. Abra o projeto no TIA Portal.
2. Na árvore do projeto, acesse External source files.
3. Clique em Add new external file e selecione o arquivo baixado (*.scl).
4. Clique com o botão direito sobre arquivo recém-adicionado e selecione Generate blocks from source.
5. O TIA Portal irá gerar o FB, que poderá ser utilizado em seus projetos.
